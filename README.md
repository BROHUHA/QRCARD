## What is This?

**QR Card** is a free, open-source tool that lets you generate printable cards with QR codes. It's perfect for:

*   **Home**: Give guests instant WiFi access without reading out passwords.
*   **Business**: Share your website or portfolio link professionally.
*   **Social**: Grow your following on Instagram, WhatsApp, TikTok, and more.

---

## Key Features

*   **Neobrutalist UI**: A high-contrast, trendy design language that stands out.
*   **Three Card Types**:
    *   **WiFi Card**: Connect instantly to WPA/WEP networks.
    *   **URL Card**: Share websites, portfolios, or payment links.
    *   **Social Card**: Custom templates for WhatsApp, Instagram, Telegram, Snapchat, and more.
*   **Customization**:
    *   Portrait or Landscape orientation.
    *   Hide/Show passwords or details.
    *   Platform-specific branding colors.
*   **Print & Download**:
    *   **Download HD**: Save as a high-quality PNG image.
    *   **Print Mode**: Optimized layout for direct printing.
*   **Privacy First**: **No data is sent to any server.** All QR generation happens locally in your browser.

## Tech Stack

Built with modern web technologies for speed and reliability:

*   **React** - Frontend Library
*   **Evergreen UI** - Component Framework
*   **React i18next** - Internationalization
*   **qrcode.react** - QR Code Generation
*   **Node-to-Image** - HD Export

## Running Locally

Want to run this yourself?

1.  **Clone the repo**
    ```bash
    git clone https://github.com/1999azzar/wifi-card.git
    cd wifi-card
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the app**
    ```bash
    npm start
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
